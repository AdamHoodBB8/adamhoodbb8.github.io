Bongles

This was a 3D point and click adventure game I created in a small team. I was originally in charge of Puzzle and Level Design but slowly transitioned into also working with the programming team to create the movement system and code the interactions and inventory systems. I was also responsible for importing art created for the game, aswell as importing animations.

This was a really great experience as the team worked well together, and we ended up creating something I'm proud of.

Honours Project

For my University Honours Project, I was looking at player type classifications (Bartle mainly) and how they correlate with that players choices in a narrative based game.

This is the short game I created that was used as part of the study. It's quite a simple game but helped me gather data in the testing phase.
